## Inicializar o projeto

```bash
# instalar pacotes
$ npm install

# iniciar container do banco de dados (postgres)
$ docker compose up -d
```


## Compilar e rodar o projeto

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Rota do Swagger

```bash
# Swagger
$ http://localhost:3000/api

```
